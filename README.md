# pygame-snake
simple typical snake game based on pygame framework.


## Play it
clone repository:
```
git clone https://github.com/rdilare/pygame-snake.git
```

```
cd pygame-snake
```

requirements(prefere using a virtual-environment):
```
pip3 install -r requirements.txt
```

run the game 
```
python3 app.py
```
